Drop your images here using EXACTLY these filenames:

  logo.png      → appears in the top navbar (recommended: transparent background, any size)
  favicon.png   → appears as the browser tab icon (recommended: square, 64×64 or larger)

Both files are optional — if a file is missing, the
site falls back to the default purple SVG logo.

Tips:
  • PNG with transparent background works best for logo.png
  • A simple square icon works best for favicon.png
  • After adding files, just refresh the browser tab
